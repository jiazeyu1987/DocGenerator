# DOCX 模板说明

## 如何添加模板

### 方法一：手动创建推荐模板

您可以使用Microsoft Word创建以下模板文件：

#### 1. 基础模板 (basic_template.docx)
- 字体：宋体 11pt
- 页边距：普通
- 适用于：一般文档、笔记、说明文档

#### 2. 技术文档模板 (tech_template.docx)
- 字体：Consolas 10pt（代码）/ 微软雅黑 11pt（说明）
- 页边距：左右1.25英寸
- 适用于：技术规范、API文档、代码注释

#### 3. 商务文档模板 (business_template.docx)
- 字体：宋体 12pt（正文）/ 黑体（标题）
- 页边距：左右2英寸，上下1.5英寸
- 适用于：合同、提案、计划书

#### 4. 学术论文模板 (academic_template.docx)
- 字体：Times New Roman 12pt
- 页边距：左右1.5英寸，上下1英寸
- 适用于：论文、研究报告、学术文档

### 方法二：使用现有模板

1. 在Word中打开一个新文档
2. 设置好您想要的样式和格式
3. 保存为 `.docx` 格式
4. 将文件复制到这个 `templates_store` 目录
5. 重启后端服务

### 方法三：下载预设模板

您可以访问以下链接下载一些常用的DOCX模板：
- [Microsoft Office 官方模板](https://templates.office.com/)
- 或者直接从网络搜索 "Word document template"

## 注意事项

- 文件名只能包含：字母、数字、点(.)、下划线(_)、连字符(-)
- 文件扩展名必须是 `.docx`
- 建议文件名使用英文字符
- 添加新模板后需要重启后端服务

## 当前可用模板

重启后端服务后，模板将出现在前端的下拉菜单中。
